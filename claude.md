### About HeroUI
HeroUI (formerly NextUI) is a modern React component library for building beautiful and performant user interfaces. It provides:
- Comprehensive pre-built React components
- Modern, visually appealing design
- Performance optimization
- Well-documented with interactive examples
- Active development and community support

### Installation
```bash
npm install @heroui/react@canary
```

### Usage Guidelines
When building UI components for this project:
1. Use HeroUI components as the primary UI framework
2. Follow HeroUI's design patterns and conventions
3. Reference the official documentation for component APIs
4. Check Storybook for interactive examples
