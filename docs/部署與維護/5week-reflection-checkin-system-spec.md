## 系統概述

### 目標
為「5週復盤習慣養成陪跑班」學員提供簡單易用的每日打卡系統，透過同儕見證與視覺化激勵，協助學員建立持續35天的復盤習慣。

### 核心功能
1. 學員每日打卡記錄
2. 即時查看其他同學打卡狀態
3. 連續打卡天數追蹤與排行
4. 每日亮點分享與交流
5. 個人打卡數據查詢

### 設計原則
- **低門檻**: 使用學員熟悉的 Google 工具
- **高可見性**: 讓學員看到彼此的進度
- **儀式感**: 透過視覺化增強打卡體驗
- **可維護**: 系統簡單穩定，維運成本低

## Part 1: Google Form 規格

### 表單名稱
**5週復盤陪跑班 - 每日打卡**

### 表單欄位設計

#### 1. 姓名 *必填*
- **類型**: 下拉選單
- **選項**: 所有學員名單 (預先建立)
- **用途**: 識別打卡學員
- **驗證**: 必填

#### 2. 打卡日期 *必填*
- **類型**: 日期
- **預設值**: 今天 (自動帶入)
- **用途**: 記錄打卡時間
- **驗證**: 必填

#### 3. 今天是否完成間歇式日記? *必填*
- **類型**: 單選
- **選項**:
  - ✅ 是，已完成
  - ⏰ 晚點補
  - ❌ 今天跳過
- **用途**: 核心打卡狀態
- **驗證**: 必填

#### 4. 今日一句話亮點 *必填*
- **類型**: 簡答
- **說明文字**: 「用一句話分享今天記錄或萃取的最大收穫 (50字內)」
- **字數限制**: 50字
- **用途**: 促進反思與同儕交流
- **驗證**: 必填，50字以內

#### 5. 今天使用了哪種萃取法?
- **類型**: 單選
- **選項**:
  - 📝 ORID 情緒萃取
  - 🎯 PAR 工作萃取
  - 📸 相片簿生活萃取
  - 🎙️ AI Podcast 訪談
  - ⏳ 只記錄，尚未萃取
- **用途**: 追蹤學員使用的方法論
- **驗證**: 選填

#### 6. 想對同學說的話
- **類型**: 段落 (長文字)
- **說明文字**: 「(選填) 鼓勵、求救、分享心情都歡迎」
- **用途**: 增加社群互動
- **驗證**: 選填

### 表單設定

- **提交限制**: 每天只能提交一次 (透過 Google 帳號限制)
- **提交後訊息**: 「✅ 打卡成功！繼續保持，你已經累積 [X] 天了！」
  - 註: 需使用 Google Apps Script 客製化動態顯示累計天數
- **收集電子郵件**: 是 (用於身份驗證)
- **開放時間**: 課程期間 24 小時開放

### 表單外觀

- **主題色**: 知識複利品牌色
- **標題圖片**: 課程視覺主視覺
- **描述文字**: 
  ```
  📝 每日 10 分鐘，記錄你的間歇式日記
  🔥 35 天養成習慣，我們一起見證彼此的成長！
  
  打卡完成後，記得到儀表板看看其他同學的精彩分享 👉 [儀表板連結]
  ```

#### Sheet 3: 打卡統計 (核心)

**用途**: 計算每位學員的打卡數據與里程碑達成狀況

**欄位設計**:
| 欄位 | 類型 | 計算邏輯 | 說明 |
|-----|------|---------|------|
| A: 姓名 | 文字 | 從「學員名單」抓取 | 學員識別 |
| B: 累計打卡天數 | 數字 | COUNTIFS | 統計該學員所有「✅ 是，已完成」的次數 |
| C: 連續打卡天數 | 數字 | 自訂公式 | 從最近打卡日往前推算連續天數 |
| D: 最近打卡日期 | 日期 | MAXIFS | 該學員最後一次打卡日期 |
| E: 7天達成 | 文字 | IF | 連續≥7天顯示「🏆」，否則「-」 |
| F: 14天達成 | 文字 | IF | 連續≥14天顯示「🏆」，否則「-」 |
| G: 21天達成 | 文字 | IF | 連續≥21天顯示「🏆」，否則「-」 |
| H: 35天達成 | 文字 | IF | 連續≥35天顯示「🏆」，否則「-」 |

**關鍵公式**:

**B欄 (累計打卡天數)**:
```excel
=COUNTIFS('表單回應'!$B:$B, $A2, '表單回應'!$D:$D, "✅ 是，已完成")
```

**C欄 (連續打卡天數)**:
```excel
// 需使用 Google Apps Script 自訂函數
// 或使用複雜的 QUERY + FILTER 組合
// 邏輯: 從最近日期往回找，連續有打卡記錄的天數
```

**D欄 (最近打卡日期)**:
```excel
=MAXIFS('表單回應'!$C:$C, '表單回應'!$B:$B, $A2, '表單回應'!$D:$D, "✅ 是，已完成")
```

**E欄 (7天達成)**:
```excel
=IF(C2>=7, "🏆", "-")
```

**排序**: 依「連續打卡天數」降冪排序

---

#### Sheet 4: 每日亮點牆 (供儀表板讀取)

**用途**: 整理所有學員的亮點分享，供 Vibe 儀表板讀取

**欄位設計**:
| 欄位 | 類型 | 資料來源 | 說明 |
|-----|------|---------|------|
| A: 日期 | 日期 | 表單回應!C | 打卡日期 |
| B: 姓名 | 文字 | 表單回應!B | 學員姓名 |
| C: 一句話亮點 | 文字 | 表單回應!E | 今日一句話亮點 |
| D: 萃取法 | 文字 | 表單回應!F | 使用的萃取方法 |
| E: 額外分享 | 文字 | 表單回應!G | 想對同學說的話 |

**資料來源公式**:
```excel
=QUERY('表單回應'!A:G, 
  "SELECT C, B, E, F, G 
   WHERE D = '✅ 是，已完成' 
   ORDER BY C DESC", 
  1)
```

**說明**:
- 只顯示「已完成」的打卡記錄
- 依日期降冪排序 (最新在最上面)
- 自動更新，無需手動維護

### 數據讀取方式

**方法 1: Google Sheet 轉 CSV**
```javascript
// 將 Google Sheet 公開連結轉為 CSV 格式
const sheetUrl = 'https://docs.google.com/spreadsheets/d/SHEET_ID/edit#gid=0';
const csvUrl = sheetUrl.replace('/edit#gid=', '/export?format=csv&gid=');

// 使用 Fetch API 讀取
fetch(csvUrl)
  .then(response => response.text())
  .then(csv => {
    // 解析 CSV 並處理資料
  });
```

**方法 2: Google Apps Script Web App**
- 建立 Apps Script 將 Sheet 資料轉為 JSON API
- Vibe 儀表板呼叫 JSON API
- 優點: 更靈活，可自訂回傳格式

**更新頻率**:
- 設定 `setInterval()` 每 5 分鐘自動刷新資料

function sendDailyReminder() {

### 附錄 B: 測試資料產生器

為了測試系統，可以用 Google Apps Script 產生假資料:

```javascript
function generateTestData() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('表單回應');
  const students = ['王小明', '李小華', '張小美', '陳大偉', '林小芳'];
  const methods = ['📝 ORID', '🎯 PAR', '📸 相片簿', '🎙️ Podcast', '⏳ 尚未萃取'];
  
  // 產生過去 10 天的假資料
  for (let i = 0; i < 10; i++) {
    const date = new Date();
    date.setDate(date.getDate() - i);
    
    students.forEach(student => {
      if (Math.random() > 0.2) { // 80% 機率打卡
        sheet.appendRow([
          date,
          student,
          date,
          '✅ 是，已完成',
          `測試亮點 ${i}`,
          methods[Math.floor(Math.random() * methods.length)],
          ''
        ]);
      }
    });
  }
}
```

### 附錄 C: 常見問題 FAQ

**Q1: 如果忘記打卡可以補打嗎？**
A: 可以，在表單中選擇正確的日期即可補打卡。但建議養成當天打卡的習慣。

**Q2: 打卡時間有限制嗎？**
A: 沒有，24 小時都可以打卡。但建議每天晚上睡前完成，形成固定儀式。

**Q3: 儀表板多久更新一次？**
A: 儀表板每 5 分鐘自動刷新一次資料，基本上可以即時看到其他同學的打卡狀態。

**Q4: 如果不小心重複提交表單怎麼辦？**
A: Google Form 設定為每天只能提交一次，如果重複提交會覆蓋當天的記錄。

**Q5: 可以修改已經提交的打卡內容嗎？**
A: 目前無法直接修改，如需修改請聯繫講師手動調整後台資料。
